# Combat Lan House
Este projeto foi uma idéia e criação minha, com o intuito de preservar a marca Combat e conceder a ela uma conexão especial com o cliente, aproximando-o e também trazendo novos clientes que desconhecem os serviços prestados pela mesma.

O site é bem discreto e responsivo, onde o cliente tem acesso a um menu para direciona-lo por todo o site, em seguida uma esteira dos serviços mais utilizados na loja,
a parte de notícias onde está reservadamente recebendo as notícias postadas pela cliente (quando logada no site como administradora);
em seguida uma parte destinada aos serviços feitos pela loja, com uma opção de pesquisar serviço, e botões para mostrar + serviços ou recolhe-los (mostra no mínimo 5)
também tem o painel de administração, para que a cliente possa alterar, editar ou excluir os serviços que desejar;
Logo em seguida, temos uma parte destinada a avaliações para o cliente avaliar a loja,
e um breve carrocel de avaliações que também são feitas de acordo com a vontade da cliente, ela pode adicionar, editar ou excluir (quando logada como administradora)
e por ultimo nosso rodapé, com os créditos, horário de funcionamento, contato e localização da loja.

## Tecnologias utilizadas
### Frontend:
- React
- TypeScript
- TailwindCSS
- Context API
### Backend:
- Node.Js / Express
- Prisma ORM
- JWT / Multer
### Infra & DB
- PostgreSQL
- Render / Cloudinary
- Git / Github

O site está em desenvolvimento, e estou trabalhando nele diariamente com a dona atual da loja e minha cliente. O site foi hospedado gratuitamente nas plataformas:
- Git / Github - aqui está o projeto com ambas as partes, frontend e backend;
- Railway - onde injetei o banco de dados;
- Vercel - onde está sendo hospedado o frontend;
- Render - aqui estou hospedando o backend do site;
- Cloudinary - aqui é onde está sendo carregada as imagens que estão recebendo upload no site, nas Notícias, Serviços e Avaliações
